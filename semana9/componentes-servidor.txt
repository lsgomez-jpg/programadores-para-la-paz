Sistema operativo:
Es el software principal que administra los recursos del computador o servidor. Permite ejecutar aplicaciones y controlar el hardware.

Node.js:
Es un entorno de ejecución que permite ejecutar código JavaScript fuera del navegador. Se utiliza para crear aplicaciones y servidores.

Aplicación server.js:
Es el archivo que contiene el código del servidor. Define las rutas y respuestas que recibirán los usuarios.

Puerto:
Es un punto de comunicación que permite al servidor recibir solicitudes. En esta práctica se utiliza el puerto 3000.

Logs:
Son registros de eventos y actividades del sistema. Ayudan a monitorear el funcionamiento y detectar errores.
